## 1. 변수
* GOlang은 신기하게 var 라는 키워드를 통해서 이것이 __변수__ 임을 선언을 먼저한다.  
* var 키워드 뒤에 변수명을 적고, 그 뒤에 변수 타입을 적는다.
* := 키워드를 사용해서 그냥 위를 쌩까고 할 수 있다. 근데 이건 func 안에서만 사용가능  

### 용례
```go 
var i, j, k int = 1, 2, 3 //한번에 여러개씩도 선언 가능
```  
```go
i := 1  //이게 var i int = 1 이랑 똑같음. 
```
